# C++ Programs

## Overview

This repository contains a collection of C++ programs demonstrating various concepts, algorithms, and problem-solving techniques. Each program is organized for ease of understanding and learning.

To compile and run any program:

```bash
g++ filename.cpp -o filename
./filename
```

You may need to install [GCC](https://gcc.gnu.org/) or [Clang](https://clang.llvm.org/) if not already done.
